# CleanTrackAI 🚀

## Building a Cleaner, Smarter Bharat

CleanTrackAI is an AI-powered civic issue reporting platform that enables citizens to report public issues such as garbage dumps, drainage blockages, broken streetlights, and sanitation concerns. The platform promotes transparency, accountability, and faster resolution through AI-assisted verification and real-time complaint tracking.

## Features

* 🤖 AI-Based Complaint Verification
* 📍 GPS-Enabled Issue Reporting
* 📸 Image-Based Complaint Submission
* 📝 Real-Time Complaint Tracking
* 👮 Officer Management Dashboard
* ✅ Resolution Proof Upload
* 📊 Heatmap Analytics
* 📢 Social Impact Dashboard
* 🔍 Complaint Status Monitoring

## Project Workflow

1. Citizen reports an issue with image and location.
2. AI verifies the complaint.
3. Complaint is stored and assigned a unique ID.
4. Officers review the complaint.
5. Resolution proof is uploaded after action is taken.
6. Status is updated and visible to citizens.
